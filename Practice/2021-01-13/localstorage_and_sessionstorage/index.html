Local Storage

  • Persistant over refreshes, tabs(close/open), browser restarts, os reboots.
  • Shared between all tabs and windows with same origin
  • Does not expire. Only gets deleted if done manually

Session Storage

  • Persistant over page refreshes
  • But expires on tab close
  • Not shared between tabs and windows
  • Every tab has new instance of storage
  • But is shared between iframes(from same origin) in the same tab
